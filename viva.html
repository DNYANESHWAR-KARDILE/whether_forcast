Web Development Viva Questions & Answers

HTML & CSS

1. What is HTML?

HTML is used to create the structure of web pages.

2. Difference between HTML and HTML5?

HTML5 is the latest version and supports audio, video, canvas, and semantic tags.

3. What are semantic tags in HTML5?

Semantic tags describe the meaning of content.

4. Explain the structure of an HTML document.

HTML document contains html, head, title, and body tags.

5. Difference between id and class?

id is unique while class can be used multiple times.

6. What is CSS?

CSS is used to style and design websites.

7. Difference between inline, internal, and external CSS?

Inline CSS is inside tag, internal is inside style tag, external is separate CSS file.

8. What is Box Model?

Box model consists of content, padding, border, and margin.

9. Difference between margin and padding?

Margin is outside spacing while padding is inside spacing.

10. What is responsive web design?

Responsive design makes websites work on all devices.

11. What is Bootstrap?

Bootstrap is a CSS framework for responsive design.

12. Difference between display:none and visibility:hidden?

display:none removes element while visibility:hidden hides only visibility.

13. What is Flexbox?

Flexbox is a CSS layout system for alignment.

14. Difference between relative and absolute positioning?

Relative positions from itself while absolute positions from parent.

15. What are media queries?

Media queries are used for responsive design.

JavaScript

1. What is JavaScript?

JavaScript makes web pages interactive.

2. Difference between var, let, and const?

var is function scoped, let is block scoped, const cannot change.

3. What is DOM?

DOM represents HTML elements as objects.

4. What is event handling?

Event handling responds to user actions like click and submit.

5. Difference between == and === ?

== compares values while === compares value and datatype.

6. What are functions?

Functions are reusable blocks of code.

7. What is an arrow function?

Arrow function is short syntax of function.

8. What is an array?

Array stores multiple values.

9. What is JSON?

JSON is used for data exchange.

10. What is form validation?

Form validation checks user input before submission.

11. Purpose of addEventListener()?

It attaches events to elements.

12. What are loops?

Loops repeat code multiple times.

13. Difference between synchronous and asynchronous?

Synchronous runs one by one while asynchronous runs without waiting.

14. What is AJAX?

AJAX updates webpage without refreshing.

15. What is local storage?

Local storage stores data in browser.

PHP & Database

1. What is PHP?

PHP is a server-side scripting language.

2. Difference between client-side and server-side?

Client-side runs in browser while server-side runs on server.

3. How to connect PHP with MySQL?

Using mysqli_connect() function.

4. What is XAMPP/WAMP?

XAMPP/WAMP are local server software.

5. What is session in PHP?

Session stores user data across pages.

6. Difference between GET and POST?

GET shows data in URL while POST hides data.

7. What is database?

Database stores organized data.

8. What is MySQL?

MySQL is a database management system.

9. What is primary key?

Primary key uniquely identifies records.

10. What is SQL?

SQL is used to manage databases.

11. Query to display all records?

SELECT * FROM table_name;

12. What is CRUD?

CRUD means Create, Read, Update, Delete.

13. Use of mysqli_connect()?

It connects PHP with MySQL.

14. What is form handling in PHP?

Processing form data using PHP.

15. Purpose of cookies?

Cookies store small user data in browser.

General Web Technology

1. What is a web browser?

Software used to access websites.

2. Difference between static and dynamic website?

Static website has fixed content while dynamic changes content.

3. What is URL?

URL is web address of website.

4. What is HTTP and HTTPS?

HTTPS is secure version of HTTP.

5. What is web server?

Web server stores and delivers web pages.

6. What is hosting?

Hosting stores website files on internet.

7. What is DNS?

DNS converts domain into IP address.

8. What is API?

API allows communication between applications.

9. Explain client-server architecture.

Client sends request and server sends response.

10. What is full stack development?

Full stack includes frontend, backend, and database.
